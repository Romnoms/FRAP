# FRAP Data Repository

Shared FRAP (Fluorescence Recovery After Photobleaching) data and analysis for the lab.

## Directory Structure

- **raw_data/** - Original microscopy files (.lif, .tif, etc.)
- **analysis_outputs/** - Processed data, CSVs, plots, and figures
- **scripts/** - Analysis scripts (Python, ImageJ macros, R, etc.)

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/Romnoms/FRAP.git
   ```

2. Add your data to the appropriate folders

3. Commit and push changes:
   ```bash
   git add .
   git commit -m "Add [description of your data]"
   git push
   ```
